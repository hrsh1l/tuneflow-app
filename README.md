# TuneFlow 🎸

A Guitar Tuner-style web app built with React, enhanced with AI-generated feedback to help users tune their instrument more accurately.

## Team
Harshil Patel
Hugo Colaco

## Features
- Real-time pitch detection
- String tuning feedback (sharp/flat/in tune)
- AI-generated tuning advice

## Tech Stack
- React
- Web Audio API
- Generative AI

## How to Run
npm install
npm start
